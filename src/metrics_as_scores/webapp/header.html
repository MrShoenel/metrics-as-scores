Metrics As Scores v__VERSION__, https://github.com/MrShoenel/metrics-as-scores

This application was conceived to accompany the paper entitled "Contextual Operationalization of Metrics As Scores: Is My Metric Value Good?" [1]. In that paper, the Qualitas.class corpus [2][3] was used to transform metrics into scores. This application, however, supports arbitrary datasets. Head over to the repository to find out which datasets are available for download.
