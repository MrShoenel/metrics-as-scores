Metrics as Scores Authors

This application accompanies the paper entitled "Contextual Operationalization of Metrics as Conventionality- and Quality Scores". Using the Qualitas.class corpus [1][2], metrics are transformed into scores, within different contexts. It allows to check metrics values of own applications against the contextual distributions as shown below. In a greater sense, it can answer the question "Is my metric value good or bad?", if either the type of metric has a built-in meaning of "good" or "bad" or if you choose one explicitly using a transformation type. For conventionality scores such as TLOC, MLOC, NOM or NSC, this can indicate how common or extreme a metric's value is.
